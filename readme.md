# Nginxctl - Terminal UI Utility for Nginx

Nginx Config is always been a tedious tasks, finding out paths, editing configs via vim, slight mistake and your deployment is messed up

To solve this issue, here i have came up with a simple TUI based utility that helps individual to setup reverse proxy in Nginx with just few clicks.
